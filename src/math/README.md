# Bullhorn Handlebars Math Helpers

## Add

**Usage:**

    {{add 5 2}}
    // Output => 7

**Other Dependencies:** N/A

## Ceil

**Usage:**

    {{ceil 5.1}}
    // Output => 6

**Other Dependencies:** N/A

## Divide

**Usage:**

    {{divide 6 3}}
    // Output => 2

**Other Dependencies:** N/A

## Floor

**Usage:**

    {{floor 5.7}}
    // Output => 5

**Other Dependencies:** N/A

## Multiply

**Usage:**

    {{multiply 5 2}}
    // Output => 10

**Other Dependencies:** N/A

## Round

**Usage:**

    {{round 5.7}}
    // Output => 6

**Other Dependencies:** N/A

## Subtract

**Usage:**

    {{subtract 5 2}}
    // Output => 3

**Other Dependencies:** N/A
